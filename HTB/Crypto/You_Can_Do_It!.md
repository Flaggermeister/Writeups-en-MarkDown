# You Can Do It!
*The flag is in the format HTB{plaintext}*
Nos dan el siguiente ciphertext:
```
YHAOANUTDSYOEOIEUTTC!
```
Por el tamaño y el charset debe ser un cifrado alfabetico de sustitución, probando el CesarBox con tamaño 7 obtenemos un mensaje legible
```
YOUSEETHATYOUCANDOIT!
```
Flag: **HTB{YOUSEETHATYOUCANDOIT!}**
